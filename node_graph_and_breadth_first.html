User-Defined Grid:

Supply a . for open blocks, a # for closed blocks, an A for the start block, and a B for the destination block.

Note: There is no format validation on this whatsoever; if it's not in the exact required format, expect strange and interesting errors.

Note2: We do detect a failure to navigate the maze, via a Depth-First search.

Here's an example grid:

##########
#A...#...#
#.#.##.#.#
#.#.##.#.#
#.#....#B#
#.#.##.#.#
#....#...#
##########
